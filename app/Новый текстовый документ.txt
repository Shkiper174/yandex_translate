Ссылка на github - https://github.com/Shkiper174/yandex_translate.git
Для запуска приложения необходим сервер, в моем случае использован node.js (https://nodejs.org/en/).

После клонирования файлов с репозитория, из консоли необходимо перейти в папку с проектом и запустить несколько комманд:
-npm install
-bower install

После чего локально установятся необходимые библиотеки.
По команде gulp, запустится локальный веб-сервер.

Определение текста проходит автоматически при нажатии на кнопку "перевести".
Конечный язык перевода установлен Русский.
